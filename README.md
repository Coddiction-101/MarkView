# MarkView — Minimal Markdown Converter (MVP)

A lightweight, real-time Markdown-to-HTML converter built using only **HTML, CSS, and JavaScript**.
This project focuses on understanding core frontend concepts like layout design, event handling, and basic parsing logic.

## 📌 Features (MVP Scope)

* Live typing preview
* Split-screen layout (Editor + Preview)
* Clean and minimal UI
* Basic Markdown support (planned):

  * Headings (`#`, `##`, `###`)
  * Bold (`**text**`)
  * Italic (`*text*`)
  * Lists (`- item`)
  * Paragraphs

---

## 🧱 Tech Stack

* **HTML** — Structure
* **CSS** — Layout and styling (Flexbox, dark theme)
* **JavaScript (Vanilla)** — Logic and Markdown parsing

## ⚙️ How It Works

1. User writes Markdown in the editor (`textarea`)
2. JavaScript captures input events
3. Markdown text is processed (parsed)
4. Converted HTML is rendered in the preview section
